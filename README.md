🚀 StudyFlow AI

AI-Powered Learning & Productivity Platform for Students

📌 项目简介

StudyFlow AI 是一个面向大学生的智能学习与内容生成平台，基于大模型能力，帮助用户高效完成学习任务与内容创作。

项目聚焦于真实学习场景，提供从“任务输入 → 内容生成 → 优化完善”的完整闭环体验，提升学习效率与输出质量。

✨ 核心功能
🧠 1. AI内容生成
📄 自动生成课程报告 / 作业初稿
📊 一键生成 PPT 大纲 + 内容填充
✍️ 学术内容结构化输出（更符合课程要求）
⚡ 2. 学习效率工具
📚 PDF / 课件智能总结
🗂️ 笔记整理与知识结构化
🧾 重点提取 & 复习卡片生成
🤖 3. 智能 Agent（任务自动执行）

输入一个学习目标，例如：

“完成一份市场营销课程报告”

系统将自动执行：

选题建议
大纲生成
内容填充
修改与优化建议
🏗️ 技术架构
LLM：MiMo（主模型）
Framework：LangChain / LlamaIndex
Backend：Python (FastAPI)
Frontend：React
Database：PostgreSQL / Redis
RAG：基于向量数据库的知识检索
🔍 核心技术亮点
🧩 多步骤 Agent Workflow（任务拆解 + 执行）
